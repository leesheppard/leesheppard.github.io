# Requirements
Built using Jekyll

# Install
```
$ git clone git@github.com:leesheppard/leesheppard.github.io.git
$ cd git@github.com:leesheppard/leesheppard.github.io.git
$ bundle install
```

To upgrade run:

```
$ bundle update github-pages
```

# How to run the app

Run Jekyll with:

```
$ bundle exec jekyll serve
```
